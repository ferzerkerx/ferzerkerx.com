Git Greenseer

Mon, Jan 1, 0001
https://github.com/ferzerkerx/git-greenseer/

A static git repository tool analyzer. It calculates the total contribution per category / project by developer and it generates a report displaying charts with this information. It helps to understand which developers own or understand specific categories on the code. Html report shows: Contribution percentage by category. (shows a bar chart) Contribution per developer for each category. (shows a pie chart)
